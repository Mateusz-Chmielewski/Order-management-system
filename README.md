## Connection data
In connection directory you must create connection_data.php where you enter connection options:

### The contents of the connection_data.php file:
<?php
    $db_host = "ServerName";
    $db_name = "DatabaseName";
    $db_user = "UserName";
    $db_password = "Password";
?>
